{# Embed-mode template — NOT extending base.html on purpose. Standalone minimal document designed to live inside an iframe on external sites (the riskmarketnews.com Ghost homepage being the first known consumer). Everything that's not directly needed to render the Signal Tape band has been stripped: * No OG / Twitter meta — embed gets indexed via the host page, not standalone. noindex below makes that explicit. * No Plausible — the dashboard tracks its own pageviews; the embed's analytics belong to the host site. * No Leaflet / Plotly / HTMX — Signal Tape is pure SVG. * No header / footer / RMN nav chrome — host page provides its own surrounding context. * Transparent body background so the iframe blends with whatever the host page paints behind it. Reuses the production /static/js/signal_tape.js loader and the /api/v1/signal-tape JSON endpoint. Zero new business logic; the mount div ID matches what the loader expects, so any future improvement to the dashboard's Signal Tape ships here automatically. Iframe sizing: the host iframe element sets the box; this document lets its content flow inside it. The Signal Tape band is a fixed-height affair (~120px desktop), and the recommended iframe markup hardcodes that, so we don't need cross-origin postMessage auto-resize gymnastics. #} {# Brand fonts — same Inter + Source Serif 4 as base.html so the Signal Tape text in the embed renders identically to its in-dashboard counterpart.
